### TC-004: Verify form submission prevents page reload and displays alert with both values

- **Priority:** Critical
- **Type:** Happy Case
- **Preconditions:** LoginForm component is rendered.
- **Test Data:** Username: `user1`, Password: `pass1`

| # | Step | Expected Result |
|---|------|-----------------|
| 1 | Type `user1` into the Username field. | The username input displays `user1`. |
| 2 | Type `pass1` into the Password field. | The password input displays masked characters. |
| 3 | Click the "Login" button. | The page should NOT reload. An alert should display: `Username: user1, Password: pass1`. |

- **Postconditions:** Form submission is prevented, and the alert shows both entered values.
