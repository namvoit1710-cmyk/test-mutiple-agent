# Manual Test Cases: SA-871 / T4 [FE]

Source spec: `app/docs/spec/SA-871.md`
Source plan task: `T4` in `app/docs/plan/SA-871.md`

### TC-001: Verify "Submit" button is disabled when field is empty

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is empty.
- **Test Data:** Empty string

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible and empty. |
| 2 | Observe the "Submit" button. | The "Submit" button is disabled. |
| 3 | Click outside the input field (trigger onBlur). | The "Submit" button remains disabled. |

- **Postconditions:** "Submit" button is disabled.

### TC-002: Verify "Submit" button is enabled when field is valid

- **Priority:** Critical
- **Type:** Happy Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is ready for input.
- **Test Data:** "Jane Smith"

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible. |
| 2 | Enter "Jane Smith" into the input field. | The text "Jane Smith" is displayed. |
| 3 | Click outside the input field (trigger onBlur). | No error message is displayed. The input field border changes to a normal/success state. |
| 4 | Observe the "Submit" button. | The "Submit" button is enabled. |

- **Postconditions:** "Submit" button is enabled.
