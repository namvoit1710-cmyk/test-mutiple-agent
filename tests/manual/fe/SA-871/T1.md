### TC-001: Verify input field rendering and basic interaction

- **Priority:** Critical
- **Type:** Happy Case
- **Preconditions:** The form page is loaded.
- **Test Data:** None

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible. |
| 2 | Observe the input field. | The label is "Full Name", placeholder is "e.g., John Doe". |
| 3 | Focus on the input field. | The input field gains focus. |
| 4 | Type a single character (e.g., "A"). | The character appears in the input field. |
| 5 | Type a second character (e.g., "B"). | The characters "AB" appear in the input field. |

- **Postconditions:** The input field contains "AB".
