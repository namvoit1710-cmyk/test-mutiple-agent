# Manual Test Cases: SA-871 / T3 [FE]

Source spec: `app/docs/spec/SA-871.md`
Source plan task: `T3` in `app/docs/plan/SA-871.md`

### TC-001: Verify validation message for empty field

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is empty.
- **Test Data:** Empty string

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible and empty. |
| 2 | Click outside the input field (trigger onBlur). | The error message "Please enter your full name." appears directly below the input field. |

- **Postconditions:** Error message is displayed.

### TC-002: Verify validation message for minimum length violation

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is ready for input.
- **Test Data:** "A"

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible. |
| 2 | Enter "A" into the input field. | The text "A" is displayed. |
| 3 | Click outside the input field (trigger onBlur). | The error message "Full name must be at least 2 characters long." appears directly below the input field. |

- **Postconditions:** Error message is displayed.

### TC-003: Verify validation message for invalid characters

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is ready for input.
- **Test Data:** "John Doe 123"

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible. |

- **Postconditions:** Error message is displayed.

### TC-005: Verify validation message for exceeding max length (if applicable)

- **Priority:** High
- **Type:** Negative Case
- **Preconditions:** The form page is loaded. The "Full Name" input field is ready for input.
- **Test Data:** A string of 51 alphabetic characters and spaces.

| # | Step | Expected Result |
|---|---|---|
| 1 | Navigate to the form page. | The "Full Name" input field is visible. |
| 2 | Enter a 51-character string into the input field. | The input field should prevent entry of the 51st character, allowing only 50 characters. |
| 3 | Click outside the input field (trigger onBlur). | An error message indicating the maximum length has been exceeded should appear (or the existing error message for invalid characters if numbers/specials were also included). |

- **Postconditions:** Error message is displayed.
