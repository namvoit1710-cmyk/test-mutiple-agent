
# Manual Test Cases: SA-875 [FE]

Source spec: `app/docs/spec/SA-875.md`

### TC-001: Redirect unauthenticated user from Home to Login

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** User is not logged in (no auth token exists). The application is running.
- **Test Data:** 

| # | Step | Expected Result |
|---|---|---|
| 1 | Open an incognito/private browser window. | The browser opens in a private session. |
| 2 | Manually navigate to the `/home` URL. | The application redirects the user to the `/login` page. |

- **Postconditions:** User is on the `/login` page and not on the `/home` page.

### TC-002: Redirect authenticated user from Login to Home

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** User is logged in (auth token exists). The application is running.
- **Test Data:** 

| # | Step | Expected Result |
|---|---|---|
| 1 | Log in to the application using valid credentials. | User is successfully logged in and an auth token is present. |
| 2 | Manually navigate to the `/login` URL. | The application redirects the user to the `/home` page. |

- **Postconditions:** User is on the `/home` page and not on the `/login` page.
