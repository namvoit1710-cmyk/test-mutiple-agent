# SA-848: Test agent - Implementation Plan

## Goal

Create a simple login form UI component in the React frontend with username and password fields. When submitted, the form should display an alert showing the values entered by the user.

## Plan

### Task 1: Create components directory [FE][simple]

**Description:** Create the `components` directory inside `app/fe/src/`.

**Action:**
1. Create the directory `app/fe/src/components`.

### Task 2: Create LoginForm.tsx component file [FE][simple]

**Description:** Create the `LoginForm.tsx` file and add the basic functional component structure, including necessary imports for React and `useState`.

**Action:**
1. Create the file `app/fe/src/components/LoginForm.tsx`.
2. Add the following initial content to `LoginForm.tsx`:
   ```typescript
   import React, { useState } from 'react';
   import './LoginForm.css'; // Will be created in the next step

   const LoginForm: React.FC = () => {
     const [username, setUsername] = useState('');
     const [password, setPassword] = useState('');

     const handleUsernameChange = (e: React.ChangeEvent<HTMLInputElement>) => {
       setUsername(e.target.value);
     };

     const handlePasswordChange = (e: React.ChangeEvent<HTMLInputElement>) => {
       setPassword(e.target.value);
     };

     const handleSubmit = (e: React.FormEvent) => {
       e.preventDefault();
       alert(`Username: ${username}, Password: ${password}`);
     };

     return (
       <div className="login-container">
         <form className="login-form" onSubmit={handleSubmit}>
           <h2>Login</h2>

           <div className="form-group">
             <label htmlFor="username">Username</label>
             <input
               type="text"
               id="username"
               value={username}
               onChange={handleUsernameChange}
             />
           </div>

           <div className="form-group">
             <label htmlFor="password">Password</label>
             <input
               type="password"
               id="password"
               value={password}
               onChange={handlePasswordChange}
             />
           </div>

           <button type="submit">Login</button>
         </form>
       </div>
     );
   };

   export default LoginForm;
   ```

### Task 4: Integrate LoginForm into App.tsx [FE][simple]

**Description:** Modify `App.tsx` to import and render the `LoginForm` component, replacing the existing "Hello World" text.

**Action:**
1. Open `app/fe/src/App.tsx`.
2. Add `import LoginForm from './components/LoginForm';` at the top of the file.
3. Replace `<h1>Hello World</h1>` with `<LoginForm />` in the `App` component's JSX.

### Task 5: Verify implementation [FE][simple]

**Description:** Run the application and verify that the login form is displayed correctly and functions as expected.

**Action:**
1. Run the development server using `npm run dev` (or `pnpm dev`).
2. Open the browser to the local URL (e.g., `http://localhost:5173`).
3. Verify that the centered login form is visible with the specified styling.
4. Type a username and password into the respective fields.
5. Click the "Login" button.
6. Confirm that an alert pops up with the format: `Username: [your_username], Password: [your_password]`.
