# Implementation Plan for SA-848: Test agent - Login Form UI Component

This plan outlines the steps to create a simple login form UI component in the React frontend, as described in the SA-848 specification.

### Task 1: Create `components` directory [FE]
Create the `components` directory inside `app/fe/src/`.

### Task 2: Create `LoginForm.tsx` [FE]
Create the `LoginForm.tsx` file inside `app/fe/src/components/`. This file will contain the React functional component for the login form.

### Task 3: Implement `LoginForm.tsx` structure and state [FE]
- Import `useState` from React and `LoginForm.css`.
- Define `username` and `password` state variables using `useState`.
- Implement `handleUsernameChange` and `handlePasswordChange` event handlers to update the respective state variables.
- Implement `handleSubmit` event handler to prevent default form submission and display an alert with the username and password values.
- Add the basic JSX structure for the login form, including the `div.login-container`, `form.login-form`, `h2`, `div.form-group` for username and password, `label`, `input`, and `button`.
- Bind the input values to the state variables and the `onChange` handlers.
- Bind the `onSubmit` handler to the form.

### Task 4: Create `LoginForm.css` [FE]
Create the `LoginForm.css` file inside `app/fe/src/components/`. This file will contain the styles for the login form.

### Task 5: Implement `LoginForm.css` styles [FE]
- Style `.login-container` to center the form on the page using flexbox and set full viewport height and a light gray background.
- Style `.login-form` as a card with a white background, padding, border-radius, box-shadow, and max-width.
- Style `.form-group` for vertical stacking of labels and inputs with margin-bottom.
- Style `label` with margin-bottom, bold font-weight, and smaller font-size.
- Style `input` fields with padding, border, border-radius, font-size, and full width.
- Style `button` with a primary background color, white text, padding, no border, border-radius, font-size, pointer cursor, full width, margin-top, and a darker background on hover.
- Style `h2` with margin-bottom, center text alignment, and a dark gray color.

### Task 6: Update `App.tsx` [FE]
- Import the `LoginForm` component at the top of `app/fe/src/App.tsx`.
- Replace the existing `<h1>Hello World</h1>` with `<LoginForm />`.

### Task 7: Verify Implementation [FE]
- Run the development server (`npm run dev` or `pnpm dev`).
- Open the browser to the local URL (e.g., `http://localhost:5173`).
- Confirm the centered login form is displayed.
- Enter values in both username and password fields.
- Click the "Login" button.
- Verify that an alert popup appears with the entered values in the format: `Username: your_username, Password: your_password`.
- Check all acceptance criteria from the specification.
