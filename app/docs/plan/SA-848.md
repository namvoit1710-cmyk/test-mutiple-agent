# Login Form Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use subagent-driven-development (recommended) or executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a simple login form UI component in the React frontend that alerts the entered username and password upon submission.

**Architecture:** A functional React component `LoginForm` using `useState` for controlled inputs, styled with a separate CSS file for a centered card layout.

**Tech Stack:** React, TypeScript, CSS.

---

## File Mapping
- Create: `app/fe/src/components/LoginForm.tsx` - The login form component.
- Create: `app/fe/src/components/LoginForm.css` - Styles for the login form.
- Modify: `app/fe/src/App.tsx` - Entry point to render the `LoginForm`.

### Task 4: LoginForm State and Logic [MEDIUM]
**Files:**
- Modify: `app/fe/src/components/LoginForm.tsx`

- [ ] **Step 1: Add state for username**
  Add the following inside the `LoginForm` component:
  ```tsx
    const [username, setUsername] = React.useState('');
  ```

- [ ] **Step 2: Add state for password**
  Add the following inside the `LoginForm` component:
  ```tsx
    const [password, setPassword] = React.useState('');
  ```

- [ ] **Step 3: Implement handleUsernameChange**
  Add the following inside the `LoginForm` component:
  ```tsx
    const handleUsernameChange = (e: React.ChangeEvent<HTMLInputElement>) => {
      setUsername(e.target.value);
    };
  ```

- [ ] **Step 4: Implement handlePasswordChange**
  Add the following inside the `LoginForm` component:
  ```tsx
    const handlePasswordChange = (e: React.ChangeEvent<HTMLInputElement>) => {
      setPassword(e.target.value);
    };
  ```

- [ ] **Step 5: Implement handleSubmit**
  Add the following inside the `LoginForm` component:
  ```tsx
    const handleSubmit = (e: React.FormEvent) => {
      e.preventDefault();
      alert(`Username: ${username}, Password: ${password}`);
    };
  ```

### Task 5: LoginForm UI [MEDIUM]
**Files:**
- Modify: `app/fe/src/components/LoginForm.tsx`

- [ ] **Step 1: Implement JSX structure**
  Replace `{/* Form will go here */}` with the following:

```tsx
      <form className="login-form" onSubmit={handleSubmit}>
        <h2>Login</h2>
        
        <div className="form-group">
          <label htmlFor="username">Username</label>
          <input 
            type="text"
            id="username"
            value={username}
            onChange={handleUsernameChange}
          />
        </div>
        
        <div className="form-group">
          <label htmlFor="password">Password</label>
          <input 
            type="password"
            id="password"
            value={password}
            onChange={handlePasswordChange}
          />
        </div>
        
        <button type="submit">Login</button>
      </form>
```
