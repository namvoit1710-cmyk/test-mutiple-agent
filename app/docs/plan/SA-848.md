# SA-848: Test agent - Implementation Plan

## Goal

Create a simple login form UI component in the React frontend with username and password fields. When submitted, the form should display an alert showing the values entered by the user.

## Implementation Plan

### Task 1: Create components directory [FE][simple]
Create the `app/fe/src/components` directory.

### Task 2: Create LoginForm.tsx [FE][simple]
Create the `app/fe/src/components/LoginForm.tsx` file with the basic functional component structure, `useState` imports, and initial state for username and password. Include placeholder event handlers and the basic JSX structure.

### Task 3: Create LoginForm.css [FE][simple]
Create the `app/fe/src/components/LoginForm.css` file and import it into `LoginForm.tsx`. Add basic styling to center the form.

### Task 4: Integrate LoginForm into App.tsx [FE][simple]
Modify `app/fe/src/App.tsx` to import and render the `LoginForm` component, replacing the "Hello World" text.

### Task 5: Implement LoginForm.tsx logic and structure [FE][medium]
Update `app/fe/src/components/LoginForm.tsx` to:
- Implement `handleUsernameChange` and `handlePasswordChange` to update state.
- Implement `handleSubmit` to prevent default form submission and display the alert with username and password values.
- Ensure the JSX structure matches the specification, including labels, input types, values, and `onChange` handlers.

### Task 6: Style LoginForm.css [FE][medium]
Update `app/fe/src/components/LoginForm.css` with detailed styling for:
- `.login-container` (flexbox for centering, full viewport height, background color)
- `.login-form` (card-style: white background, padding, border radius, box shadow, max-width)
- `.form-group` (margin-bottom, flex column)
- Labels (margin-bottom, font-weight, font-size)
- Inputs (padding, border, border-radius, font-size, full width)
- Button (background, color, padding, border, border-radius, font-size, cursor, full width, margin-top, hover state)
- `h2` (margin-bottom, text-align, color)

## Acceptance Criteria

- [ ] New `components` directory exists in `src/`
- [ ] `LoginForm.tsx` component file exists and exports a React component
- [ ] `LoginForm.css` file exists and is imported in LoginForm.tsx
- [ ] Form has a "Login" heading (h2)
- [ ] Form has a "Username" label and text input field
- [ ] Form has a "Password" label and password input field (characters hidden)
- [ ] Form has a "Login" submit button
- [ ] Typing in username field updates the component state
- [ ] Typing in password field updates the component state
- [ ] Clicking submit button prevents page reload
- [ ] Clicking submit button displays an alert with format: `Username: [value], Password: [value]`
- [ ] Form is visually centered on the page
- [ ] Form has a card-style appearance (white background, shadow, border radius)
- [ ] Input fields and button are styled and easy to use
- [ ] App.tsx imports and renders the LoginForm component
