### Task 1: Create usePrivateRouteGuard hook [FE][simple]
**Goal:** Redirect users with no auth token away from a protected page to /login.
**File:** `app/fe/src/features/auth/hooks/usePrivateRouteGuard.ts`
**Steps:**
1. Create the file and export `function usePrivateRouteGuard(): void`.
2. Inside it, read the token with `const token = localStorage.getItem('token')`.
3. Get `const navigate = useNavigate()` from `react-router-dom`.
4. In a `useEffect` with deps `[token, navigate]`, if `!token` call `navigate('/login', { replace: true })`.
