### Task 1: Create `usePrivateRouteGuard` hook [FE][simple]
**Goal:** Implement a React hook to redirect unauthenticated users away from protected routes.
**File:** `app/fe/src/features/auth/hooks/usePrivateRouteGuard.ts`
**Steps:**
1. Create the file and export `function usePrivateRouteGuard(): void`.
2. Inside the hook, read the authentication token using `localStorage.getItem('token')`.
3. Get the `navigate` function from `react-router-dom` using `useNavigate()`.
4. Use a `useEffect` hook with dependencies `[token, navigate]`. If `!token` is true, call `navigate('/login', { replace: true })`.
