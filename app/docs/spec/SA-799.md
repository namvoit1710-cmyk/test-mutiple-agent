# SA-799: Create a test Form

## Description

Title: Technical Requirement Description for login.html 

Objective:{{ }}
Create a new HTML file named login.html that includes a simple login form with two input fields: username and password. 

Requirements: 

# *File Creation* 

* Create a new file named login.html. 

# *HTML Structure* 

* The file should contain a basic HTML structure including <!DOCTYPE html>, <html>, <head>, and <body> tags. 

# *Form Elements* 

* Include a <form> element that will contain the following: 
* An input field for the username: 
* Type: text 
* Name: username 
* Placeholder: Enter your username 
* An input field for the password: 
* Type: password 
* Name: password 
* Placeholder: Enter your password 
* A submit button to submit the form: 
* Type: submit 
* Value: Login 

# *Accessibility* 

* Ensure that each input field has an associated <label> for accessibility: 
* The label for the username field should be "Username". 
* The label for the password field should be "Password". 

# *Styling (Optional)* 

* Basic CSS can be included within a <style> tag in the <head> section to improve the appearance of the form. 

# *Validation (Optional)* 

* Client-side validation can be implemented using HTML5 attributes such as required on the input fields. 

# *No External Dependencies* 

* The implementation should not rely on any external libraries or frameworks. 

Deliverables: 

* A fully functional login.html file that meets the above requirements. .
