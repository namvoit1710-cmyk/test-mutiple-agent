}

## CSS

Single BEM class `.input`:

- `border: 1px solid red`
- `border-radius: 16px`
- `padding: 8px 12px`
- `outline: none`
- `font-size: 14px`
- `width: 100%`

No additional styles. Consumer applies `className` for width control or other overrides.
