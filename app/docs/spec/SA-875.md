# SA-875: Handle authen for app

## Description

{noformat}# Auth Routing Guards — Design Spec

**Date:** 2025-06-13
**Scope:** Route protection via React Context + React Router guards

## Overview

Protect routes so unauthenticated users cannot access `/home`, and authenticated users cannot access `/login`. Auth state is in-memory only (no localStorage, no backend) — a simple demo via React Context.

## Architecture

```
AuthProvider (Context)
  └── BrowserRouter
        ├── /login  → AuthRoute guard  → Login page
        └── /home   → PrivateRoute guard → Home page
```

**Auth flow:** `login()` sets `isAuthenticated = true` → guard redirects kick in automatically.

## Files

| File | Purpose |
|---|---|
| `src/lib/auth-context.tsx` | `AuthProvider`, `useAuth()` — exposes `isAuthenticated`, `login()`, `logout()` |
| `src/components/private-route.tsx` | Wraps protected pages. If `!isAuthenticated` → `<Navigate to="/login">` |
| `src/components/auth-route.tsx` | Wraps auth pages. If `isAuthenticated` → `<Navigate to="/home">` |
| `src/pages/home.tsx` | Placeholder. Full UI in separate ticket. |
| `src/pages/login.tsx` | Placeholder. Full UI in separate ticket. |
| `src/App.tsx` | Updated — wraps app in `<AuthProvider>` + `<BrowserRouter>` with routes |

## Dependencies

- `react-router-dom` v6 (compatible with React 18)

## Routing Logic

### PrivateRoute (`/home`)

```
if not authenticated → redirect to /login
if authenticated     → render child (Outlet or children)
```

### AuthRoute (`/login`)

```
if authenticated     → redirect to /home
if not authenticated → render child
```

## Acceptance Criteria

| # | Scenario | Expected |
|---|---|---|
| 1 | No auth → navigate to `/home` | Redirected to `/login` |
| 2 | Authenticated → navigate to `/login` | Redirected to `/home` |
| 3 | No auth → navigate to `/login` | Page renders (stays) |
| 4 | Authenticated → navigate to `/home` | Page renders (stays) |

## Out of Scope

- Login form UI (separate ticket)
- Home page content (separate ticket)
- Persisted auth / localStorage
- Backend auth integration
