# SA-871: create form input name

## Description

h1. Requirement Specification: Name Input Form

h2. 1. Overview

A simple input form to collect the user's "Full Name", ensuring the entered data is accurate and contains no invalid characters.

h2. 2. UI Details

* *Label:* Full Name
* *Input Type:* {{text}}
* *Placeholder:* "e.g., John Doe"
* *Required:* Yes

h2. 3. Validation Rules

* *Min length:* 2 characters.
* *Max length:* 50 characters.
* *Format:* Alphabetic characters and spaces only. No numbers or special characters allowed.
* *Error Messages:*
** _If empty:_ "Please enter your full name."
** _If < 2 characters:_ "Full name must be at least 2 characters long."
** _If contains numbers/special characters:_ "Full name can only contain letters and spaces."

h2. 4. Acceptance Criteria

* [ ] The user can focus on the input field and type text normally.
* [ ] The "Submit" button (or next action) remains disabled if the field is empty or contains an error.
* [ ] An error message appears directly below the input field if the user enters invalid data and clicks outside the field ({{onBlur}} event).
* [ ] When the data is valid, the input border changes to a normal/success state, and all error messages are hidden.
