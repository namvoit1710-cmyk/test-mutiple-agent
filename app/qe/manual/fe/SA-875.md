# Manual Test Cases: SA-875 [FE]

Source spec: `app/docs/spec/SA-875.md`

### TC-001: Redirect unauthenticated user from Home to Login

- **Priority:** Critical
- **Type:** Negative Case
- **Preconditions:** User is not logged in (no auth token exists in localStorage). The application is running.
- **Test Data:**
  - URL: `/home`

| # | Step | Expected Result |
|---|------|-----------------|
| 1 | Manually navigate to the `/home` URL by typing it into the browser's address bar. | The application redirects the user to the `/login` page. |

- **Postconditions:** User is on the `/login` page.
